## How I Use This Repo / 使用方式

**English**
- Notes: put long-form notes in `notes/` (recommended), or keep them near related topics.
- Code: put experiments in `playground/` or `experiments/` (recommended).
- Each learning item should have a short title + date for easy searching.

**中文**
- 筆記：建議集中放在 `notes/`，或依主題放在對應資料夾旁。
- 程式碼：建議把實驗放在 `playground/` 或 `experiments/`。
- 每次學習內容建議用「日期 + 簡短標題」命名，方便搜尋與回顧。
