YC Todo：之後如何繼續 Improve App ＆ 正確開發 / 發佈流程（補充筆記）
============================================================

這一份是「完成第一版 macOS menubar app 之後」的實戰流程筆記。
目的：讓你之後不會混亂，也知道什麼時候該跑 dev、什麼時候該 build。

------------------------------------------------------------
一、之後要怎麼繼續改 YC Todo？
------------------------------------------------------------

你之後只會在兩種模式之間切換：

------------------------------------------------------------
【模式 1】開發模式（改功能 / 改 UI / Debug）
------------------------------------------------------------
用途：
- 加新功能（timer、鎖定、排序）
- 改 React UI / CSS
- 改 tray / menubar 行為
- Debug 問題

指令（在專案根目錄）：
  npx tauri dev

特性：
- 需要 VS Code + terminal 開著
- React 有熱更新（很快）
- Rust / Tauri 改動會重新編譯（較慢）
- 不是正式 app（只是 testing）

👉 任何功能改動，一定先用這個模式測

------------------------------------------------------------
【模式 2】正式 App（實際使用 / 發佈）
------------------------------------------------------------
用途：
- 變成真正的 macOS app
- 不需要 VS Code / terminal
- 拖進 Applications 使用

指令：
  npx tauri build

產物：
- .app：
  src-tauri/target/release/bundle/macos/YC ToDo.app
- .dmg：
  src-tauri/target/release/bundle/dmg/YC ToDo_0.1.0_aarch64.dmg

👉 每次想「用最新版本」都必須重新 build

------------------------------------------------------------
二、最重要的標準工作流程（一定要記住）
------------------------------------------------------------

你之後的開發循環永遠是：

  改功能 / 改 UI
        ↓
  npx tauri dev   （測試）
        ↓
      OK
        ↓
  npx tauri build （產生正式 app）
        ↓
  覆蓋 Applications 裡的舊 app

如果 app 看起來還是舊的：
- 確認有沒有覆蓋 Applications 裡的版本
- 必要時執行：
  killall Finder

------------------------------------------------------------
三、常見情境 → 要跑哪個？
------------------------------------------------------------

Q1：我只是在調 UI、CSS、React state
→ npx tauri dev

Q2：我在改 tray icon、倒數、focus 行為
→ npx tauri dev

Q3：我要給朋友用 / 我自己每天用
→ npx tauri build

Q4：我 build 了，但 app 沒變
→ 你忘了覆蓋 Applications 裡的 app，或被 macOS cache

------------------------------------------------------------
四、之後可以怎麼繼續升級這個 App？
------------------------------------------------------------

你現在已經完成「第一個可用版本」，接下來可以往這些方向進化：

【UX / 產品等級】
- tray icon 顯示剩餘時間（例如 12:34）
- 視窗精準貼齊 menubar icon 下方（popover 感）
- timer 結束自動通知（Notification Center）
- 專注模式：進行中 task 鎖定，其它 task 不能動

【系統整合】
- 開機自動啟動（App 內 toggle）
- Global shortcut（例如 ⌘⇧T 開關視窗）
- 深色 / 淺色模式自動切換
- Dock 永遠隱藏（你已完成）

【發佈 / 分享】
- 使用 .dmg 發給朋友
- 之後可做 code signing / notarization
- 未來可跨平台（Windows / Linux）

------------------------------------------------------------
五、一句總結（定位）
------------------------------------------------------------

YC Todo 不再只是學習專案，而是：

「一個真正可以每天使用的 macOS menubar productivity app」

你之後只要記住：
- 改功能 → dev
- 要用 / 發佈 → build
