Etherscan Check（鏈上驗證紀錄）
==============================

本次部署與鑄造完成後，透過 Etherscan（Sepolia Testnet）對 NFTVtwo 合約進行鏈上驗證。
以下為三個關鍵頁面之解讀，用以確認合約部署與 NFT mint 是否成功。

--------------------------------------------------
圖 1：Transaction Overview（交易總覽）
--------------------------------------------------

頁面標題顯示：
Mint 1 of NFTee

此頁代表：
這是一筆 ERC-721 標準的「Mint（鑄造）」交易。

重點欄位解讀如下：

1. Transaction Action：Mint 1 of NFTee  
   - 表示本次交易鑄造了 1 顆 NFT  
   - NFT 系列名稱為 NFTee，代號為 ITM  

2. From  
   - 地址：0xf47dE1c6...243B7d  
   - 為本人錢包地址  
   - 同時也是合約部署者（deployer）與合約 owner  

3. Interacted With (To)  
   - 地址：0xe3dd2de1...  
   - 為 NFTVtwo 合約地址  
   - 標示為 Create，代表此交易中「同時建立了合約」  

4. ERC-721 Tokens Transferred  
   - Token ID：1  
   - From：0x0000000000000000000000000000000000000000  
   - To：本人錢包地址  
   - 代表標準 ERC-721 mint 行為（從無到有）

結論（圖 1 證明）：
NFTVtwo 合約已成功部署，且在 constructor 中成功 mint 了 tokenId = 1 的 NFT 給部署者本人。

--------------------------------------------------
圖 2：State Differences（狀態變化）
--------------------------------------------------

此頁回答的核心問題是：
「這筆交易實際改變了哪些鏈上狀態？」

可觀察到三組狀態變化：

1. Validator / Producer  
   - ETH 餘額小幅增加  
   - 屬於正常的區塊驗證／出塊獎勵  

2. 本人錢包地址  
   - ETH 餘額減少（支付 gas fee）  
   - Nonce 從 7 增加為 8  
   - 表示成功送出一筆交易  

3. NFTVtwo 合約地址（0xe3dd2de1...）  
   - 出現 ITM（NFTee 的代號）  
   - Nonce 從 0 變為 1  
   - 代表合約首次被寫入狀態  
   - 包含 _mint 與 baseTokenURI 等狀態初始化完成  

結論（圖 2 證明）：
NFT 的所有權資料與合約內部狀態已永久寫入區塊鏈。

--------------------------------------------------
圖 3：Logs / Event Logs（事件紀錄）
--------------------------------------------------

此頁為 ERC-721 最標準、最具法律與技術效力的鏈上證據。

可觀察到兩個關鍵事件（Events）：

Event 1：OwnershipTransferred  
- previousOwner：0x0000000000000000000000000000000000000000  
- newOwner：本人錢包地址  

意義說明：
- 合約剛被部署時，owner 從「無」轉移給部署者  
- 表示本人為 NFTVtwo 合約的 owner（擁有 setBaseURI 等管理權限）

Event 2：Transfer（ERC-721 核心事件）  
- from：0x0000000000000000000000000000000000000000  
- to：本人錢包地址  
- tokenId：1  

意義說明：
- ERC-721 標準定義的 mint 行為  
- from 為 0x0 表示「非轉帳，而是鑄造」  
- tokenId = 1 正式存在於鏈上並歸本人所有  

結論（圖 3 證明）：
tokenId = 1 的 NFT 已依 ERC-721 標準合法鑄造，並成功分配給本人錢包地址。

--------------------------------------------------
總結
--------------------------------------------------

綜合以上三個頁面的鏈上證據，可確認：

- NFTVtwo 合約成功部署於 Sepolia Testnet  
- constructor 中的 _mint(msg.sender, 1) 已成功執行  
- tokenId = 1 的 NFT 已存在於鏈上  
- 合約 ownership 與 NFT 所有權皆正確歸屬本人  

Etherscan 驗證結果顯示本次 NFTVtwo 的部署與鑄造流程完整、正確且符合 ERC-721 標準。
